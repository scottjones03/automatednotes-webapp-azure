Welcome to the Automatic Note Creator! This tool allows you to generate comprehensive notes from uploaded PDFs. Simply follow the steps below to create your notes effortlessly.

How to Use the Automatic Note Creator:

  1. Enter your name and password to log in.
  2. Specify the file prefix, topic, content, and the number of revision questions.
  3. Provide a prompt to guide the note generation process.
  4. Upload the relevant PDFs.
  5. Set the start page index if necessary.
  6. Select the desired model for note creation.
  7. Click the submit button to generate your notes.
